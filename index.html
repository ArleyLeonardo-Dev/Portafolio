Sobre mi

Me interesé por la programación a mediados de 2020, empezando por el lenguaje Python y pasando temporalmente a Java. Me mantuve haciendo programas pequeños, nada sofisticados, tanto en Java como en Python, entrando al paradigma de la Programación Orientada a Objetos (POO) en ambos lenguajes. Decidí continuar con el lenguaje Python por su facilidad y rapidez.

En el año 2023, a mediados de marzo, conocí el framework de desarrollo web Django. Comencé a estudiarlo y a comprender cómo se desarrolla con él, aplicando la biblioteca de creación de APIs Django Rest Framework, creando CRUD utilizando PostgreSQL como base de datos.

Soy autodidacta y me apasiona el desarrollo web. Todos los días me mantengo en constante aprendizaje de este maravilloso mundo.
